# English(Ireland) dead key free keyboard layout
A custom English(Irish) Windows keyboard layout to type backtick with one keystroke

[Microsoft Keyboard Layout Creator (MSKLC) Version 1.4](https://www.microsoft.com/en-us/download/details.aspx?id=102134) was used to change the existing Irish (English Ireland) keyboard layout to remove (disable) a default dead key function of a backtick "`" key.
